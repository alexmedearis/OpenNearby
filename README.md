Open Nearby
=========
Open Nearby is an app for iOS that shows you cafes, liquor stores, restaurants and diners that are nearby and open nearby.

Version
-
1.1

Tech
-----------
Uses the Google Maps API

Installation
--------------
```
git clone [git-repo-url] stillopen
cd stillopen
open StillOpen.xcodeproj
```
