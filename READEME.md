StillOpen
=========
StillOpen is an app for iOS that shows you cafes, liquor stores, restaurants and diners that are nearby and still open.

Version
-
1.0

Tech
-----------
Uses the Google Maps API

Installation
--------------
```
git clone [git-repo-url] stillopen
cd stillopen
open StillOpen.xcodeproj
```
